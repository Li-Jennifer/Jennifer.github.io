# 2.1 Heap files

  1. Insert : O(1)
  2. Retrieve 检索 : O(b)
  3. Delete：O(b)+1

# 2.2 Sequential files : Ordered Files

  1. Retrieve: O(log2b)
  2. Insert: expensive, find and other need move, (chain points need update)
  3. Delete: Expensive
  4. Update: O(log2b)+O(1)
